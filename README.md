# STM32三重解锁智能门锁系统

## 项目简介
基于STM32F103C8T6开发的三重解锁智能门锁，支持密码、RFID刷卡、指纹识别三种解锁方式，解决传统门锁解锁方式单一、安全性弱的问题，实现多模式稳定解锁与本地权限管理。

## 核心功能
- 密码解锁：4×4矩阵键盘输入，编写按键消抖与密码校验逻辑，支持固定密码验证开锁
- RFID刷卡解锁：SPI驱动RC522射频模块，可录入、删除最多4张授权IC卡，卡UID存入本地进行权限匹配
- 指纹解锁：UART驱动AS608指纹模块，支持指纹录入、存储、特征比对，解锁识别准确率高
- 权限持久化：授权IC卡UID存储在STM32片内Flash，断电数据不丢失，支持随时管理卡片
- 状态管理：采用全局标志位状态机管控业务逻辑，同一时间仅运行一种模式，避免功能冲突
- 人机交互：OLED屏幕实时显示系统状态、操作提示；舵机模拟门锁开关，蜂鸣器做动作提示

## 硬件模块
- 主控：STM32F103C8T6
- 输入外设：4×4矩阵键盘、独立功能按键
- 识别外设：RC522射频模块、AS608指纹模块
- 显示外设：0.96寸OLED显示屏
- 执行外设：SG90舵机、蜂鸣器

## 通信协议
- UART：AS608指纹模块串口通信
- SPI：RC522射频模块通信
- I2C：OLED显示屏驱动通信
- PWM：TIM定时器输出控制舵机转角

## 开发环境
- 开发工具：Keil MDK
- 固件版本：STM32标准库

## 项目亮点
- 集成密码、RFID、指纹三重解锁，多场景适配，安全性更高
- 采用状态机分层管理业务逻辑，架构清晰，模式互不干扰
- 做了按键消抖、串口指令校验、舵机稳控等鲁棒性优化
- 模块化编程，驱动与业务逻辑分离，代码易移植、可二次扩展

## 项目难点与解决方案
- 按键误触发、连按问题：采用软件延时消抖，配合状态机限制仅当前模式响应按键，避免误操作
- 指纹模块串口刷屏、反复报错：规范收发时序，增加应答校验与采集延时，避免循环无效采集
- 多功能并发逻辑混乱：使用全局标志位做状态机，锁定当前运行模式，杜绝模式冲突
- 舵机抖动、定位不准：精准配置TIM输出50Hz PWM波形，固定占空比控制0°/90°开关门角度
- IC卡信息断电丢失：将授权卡UID写入STM32片内Flash，实现永久存储，支持动态增删管理
