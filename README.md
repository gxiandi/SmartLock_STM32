# STM32三重解锁智能门锁系统

## 项目简介
基于STM32F103C8T6开发的三重解锁智能门锁，支持密码、RFID刷卡、指纹识别三种解锁方式，解决传统门锁安全性单一的问题，实现了稳定可靠的多模式解锁控制。

## 核心功能
- 密码解锁：编写矩阵键盘消抖与密码验证逻辑，支持自定义密码
- RFID刷卡解锁：通过SPI驱动RC522模块，实现4张IC卡UID识别与授权管理
- 指纹解锁：开发UART驱动AS608指纹模块，实现指纹录入、比对与ID管理，解锁识别准确率达100%
- 状态管理：通过状态机管理多模式切换，避免状态冲突；授权IC卡UID存储于片内Flash，断电数据不丢失

## 技术栈
- 主控：STM32F103C8T6
- 外设：矩阵键盘、RC522射频模块、AS608指纹模块、OLED显示屏
- 协议：UART、SPI、I2C
- 开发工具：Keil MDK、STM32CubeIDE

## 项目亮点
- 三重解锁方式，安全性高，满足不同场景的使用需求
- 优化的状态机管理与按键防抖逻辑，解决按键误触发、舵机抖动问题，系统稳定性强
- 完整的驱动代码与测试工程，可直接运行，为嵌入式多模块联动开发提供实践经验
